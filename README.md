# VÖKA — Streetwear Essencial

> **Wear less. Mean more.**

Marca de streetwear minimalista unissex baseada em Maringá, PR. Importação direta de vestuário da China via 1688 + Superbuy para distribuição no Brasil e América Latina.

## Deploy da landing page

A landing page é um HTML estático. Pode ser hospedada em:

- **GitHub Pages**: Ative nas settings do repo (source: `/landing-page`)
- **Vercel**: Importe o repo, root directory: `landing-page`
- **Netlify**: Drop do arquivo `landing-page/index.html`
