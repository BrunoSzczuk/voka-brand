1. Acesse [superbuy.com](https://www.superbuy.com)
2. Cole o link do produto do 1688 na barra de busca
3. Selecione cor, tamanho e quantidade
4. Adicione ao carrinho e pague (item + frete doméstico China)

## Dicas importantes

- **Amostras primeiro**: Sempre compre 2-3 peças antes do lote grande
- **QC photos**: O Superbuy tira 3 fotos grátis de cada item. Peça fotos extras (¥2-3) para verificar detalhes
- **Consolidação**: Acumule itens no armazém (90 dias grátis) e envie tudo junto para economizar frete
- **Frete para o Brasil**: Use EMS ou Yanwen (aéreo econômico). Evite DHL/FedEx (muito caro para o volume)
- **Peso estimado**: 1 camiseta 200-300g ≈ 250-400g embalada. 100 camisetas ≈ 12-18kg

## Observações para o fornecedor

Nas observações do pedido ("Remark" ou "Note to agent"), sempre informe:

- Que quer o produto **sem etiqueta do fornecedor** (se for whitelabel)
- Se quer logo impresso, escreva: `需要印我的品牌logo，稍后发图`
- Peça para embalar individualmente: `请单独包装每件`
